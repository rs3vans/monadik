orElseThrow

inline fun <T : Any> Option<T>.orElseThrow(exception: () -> Exception): T

Terminates this Option by returning its value when it is an instance of Option.Some, otherwise throwing the exception produced by invoking the given supplier.

Parameters

T - the type wrapped by this Option, which is returned

exception - the supplier invoked if this is Option.None

Exceptions

Exception - given by the exception supplier if this is Option.None

Returns
a value of type T
